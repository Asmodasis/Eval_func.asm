This is the readme for eval_func.asm

it is a class project for CS_219 at Truckee Meadows Community College

The assignment involved creating a polynomial solver in MIPS assembly,

It takes a polynomial of order 2 and solves it with an input value,

finds the roots using quadratic formula, and Newtons method for computing roots

Run the program in MARS Mips emulator and follow the prompts.
